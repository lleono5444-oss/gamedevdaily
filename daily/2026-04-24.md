# Git 기본 흐름 (최소 정리)

## 📥 처음 가져오기

```bash
git clone 레포주소
cd 레포명
code .
```

---

## 📤 파일 올리기(수정)

```bash
git status
git add .
git commit -m "내용"
git push origin main
```
---

## 🔄 최신 상태 가져오기

```bash
git pull origin main
```
